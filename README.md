# FinPal

A polished finance assistant app UI concept focused on the welcome flow and authentication experience, with full dark mode support.

## Overview
FinPal is a UI design and frontend implementation for an AI finance assistant app. This version includes the welcome screen and authentication screens, built with a clean, modern look and support for both light and dark themes.

## Features
- Welcome screen with a simple, friendly onboarding flow
- Authentication screens for sign in and sign up
- Dark mode support
- Clean, responsive UI
- Finance-focused visual style
